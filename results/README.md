# Results

This folder is used to store model training and evaluation outputs generated by `src/train.py`.

After running the training script, the following files are saved automatically:

* `best_model.pth`
* `metrics.txt`
* `training_loss.png`
* `validation_accuracy.png`
* `confusion_matrix.png`

The `metrics.txt` file contains the best validation accuracy, classification report and confusion matrix.

The result figures are generated after model training and are not included as fixed outputs because they depend on the dataset split, training configuration and random initialization.

This project is intended as a learning and portfolio demonstration of a medical image classification workflow, not as a clinically validated diagnostic system.
